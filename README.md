# Бронирование переговорных комнат

Управление бронированиями переговорных комнат.

## Возможности

- API для создания, удаления брони.
- Возможность отката/повтора действия.
- Типы пользователей (RBAC).
- Множество политик разрешения конфликтов.
- Ресурусы в комнате для бронирования.
- CLI для запуска.

## Зависимости

- C++20
- CMake
- GoogleTest
- clang-format` + файл `.clang-format` для проверки стиля

## Сборка

```bash
cmake -S . -B build
cmake --build build
````

## Тесты и стиль

Запуск всех тестов:

```bash
ctest --test-dir build
```
